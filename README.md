# Zomato.-Dashboard
Zomato Sales Analysis Dashboard | Power BI
1. Project Overview
This project focuses on building a Zomato Sales Analysis Dashboard using Power BI to analyze and visualize sales performance data.
The dashboard converts raw transactional data into meaningful insights, enabling better understanding of business performance and customer purchasing patterns.
The primary objective of this project is to demonstrate practical skills in data analysis, data visualization, KPI tracking, and business reporting, aligned with real-world data analyst responsibilities.

2. Business Problem Statement
Zomato handles a large volume of sales transactions across multiple food categories and products. While this data is valuable, raw datasets do not directly provide insights required for effective business decision-making.
The key business challenges addressed in this project are:
Monitoring overall sales performance
Identifying top-performing and low-performing product categories
Understanding sales and quantity distribution
Presenting key metrics in a clear and interactive format
This project solves these challenges by transforming raw Zomato sales data into a structured and interactive Power BI dashboard, enabling data-driven insights for strategic and operational decisions.

3.Tools & Technologies
Power BI – Data visualization and dashboard development
SQL – Data querying and analysis
Microsoft Excel / CSV – Data source
Power Query – Data cleaning and transformation

4. Key Performance Indicators (KPIs)
The dashboard tracks the following KPIs:
Total Sales
Total Quantity Sold
Average Sales
Maximum Sales
Minimum Sales

5. Dashboard Features
Sales distribution by product category
Quantity analysis using pie charts
Identification of top-performing products
Interactive slicers and filters for dynamic analysis
Clean, business-focused dashboard layout

6. Insights & Analysis
Identified high-performing product categories
Analyzed sales contribution across different products
Evaluated overall sales performance using KPIs
Enabled quick insight generation through interactive visuals

7. Dataset Description
The dataset contains Zomato sales-related information such as:
Product categories
Sales values
Quantity sold
Data was cleaned, transformed, and modeled to ensure accuracy and consistency before visualization.

8. How to Access the Project
Download the Power BI file (.pbix)
Open the file using Power BI Desktop
Use filters and visuals to explore the dashboard
